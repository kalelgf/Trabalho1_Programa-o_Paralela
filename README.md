# Simulador de fluxo de vendas (pthreads)

Simulador simples de fluxo de vendas com pipeline e filas produtor-consumidor.

## Compilacao
make

## Execucao
./simulador

## Ajustes rapidos
Edite os #define no topo do codigo C para mudar quantidade de pedidos, threads, tamanho das filas e taxas de falha.
